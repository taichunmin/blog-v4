## 加速 input / output 技巧

* 使用 `scanf`, `printf`
* 使用 `sync_with_stdio(false);`

```cpp
int main()
{
  // 需注意這樣會有 scanf, printf 和 cin, cout 不同步的副作用
  cin.sync_with_stdio(false);
  cout.sync_with_stdio(false);
}
```

## 浮點數精確度處理

```cpp
int dzero( double da )
{
    const double delta = 1e-8;
    return ( da < -delta ) ? -1 : ( da > delta );
}
```
